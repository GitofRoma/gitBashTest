1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
   1)перейдите в папку куда копировать - команд cd .
   2)git clone (адрес репозитория)

2. Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
   1)git init                                             - создать новый git репозиторий
   2)git commit -m "комментарий" 
   3)git remote add origin (адрес репозитория на GitHub)  - связать с удаленным репозиторием
   4)git branch -M main                                   - меняет Master на Main
   5)git add (название_файла) или git add .
   6)push -u origin main

3. Как получить на локальный компьютер изменения с github.com?
   1) git pull origin master

4. Какую информацию нам дают команда :
   git status      - позволяет узнать состояние локального репозитория(выводит список файлов, которые были изменены или добавлены)
   git log         - озволяет просмотреть историю коммитов в локальном репозитории.
   git remote -v   - выводит список удаленных репозиториев, которые связаны с локальным
